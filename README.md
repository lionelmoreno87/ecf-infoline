# Projet InfoLine - Infrastructure DevOps

## Description
Ce dépôt contient le code de l'infrastructure et des applications pour le projet InfoLine (ECF Hiver 2025).

## Architecture
- **Infrastructure :** AWS (EKS + Lambda + RDS) via Terraform (IaaS).
- **Applications :** Java Spring Boot (API) et Angular (Front).
- **Supervision :** Stack ELK.
